# Exoplanet Microservice

This microservice provides functionality for managing different exoplanets.

## Features

- Add an exoplanet
- List exoplanets
- Get exoplanet by ID
- Update exoplanet
- Delete exoplanet
- Fuel estimation for a trip to an exoplanet
